## Comment utiliser cette icône

### Outil recommandé : AppIcon.co
1. Allez sur https://appicon.co
2. Téléversez l'image ci-dessus
3. Cochez "iPhone", "iPad", "Android"
4. Téléchargez le ZIP
5. Placez les fichiers dans :
   - Android : android/app/src/main/res/mipmap-*/
   - iOS : ios/App/App/Assets.xcassets/AppIcon.appiconset/
